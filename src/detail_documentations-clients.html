Backups and undo on Webflow

You can undo a change by using the cmd + z (apple users) or ctrl + z (windows and linux users) keyboard shortcut.

To redo, use cmd + shift + z (apple users) or ctrl + shift + z (windows and linux users)

Webflow also have a backup system. It will automatically create a backup at some times or when detecting changes on the website. You can also create manual backups when you want to save a particular state from your Webflow, before doing a major change for example.

You can either restore a backup or just preview it in a read-only link to copy-paste things or just see how it was.
